第二阶段：2026-04-22
## LSM写入路径全链路实现完成！ 🚀
### 已实现的核心组件：
1. SkipList（跳表）

- 基于跳表的有序数据结构
- 随机高度生成
- 支持插入、查找
- 线程安全的迭代器
2. WriteBatch（批量写）

- 原子批量操作接口
- 支持 Put 和 Delete 操作
- 迭代器接口解析批量操作
3. MemTable

- 基于 SkipList 实现
- Put/Delete/Get/Scan 操作
- 逻辑删除（墓碑标记）
- 容量控制与内存占用统计
- 线程安全（互斥锁保护）
- 内存池优化（Arena）
4. WAL（Write-Ahead Log）

- 32KB 块大小，顺序追加写入
- CRC32 校验
- 支持 Fragmented Record（Full/First/Middle/Last）
- Append 追加、Recover 恢复、Close 关闭、Sync 同步
- Truncate 清理接口
5. Immutable MemTable

- 只读不可修改版本
- 支持 Get/Scan
- 为 Flush 到 SSTable 做准备
